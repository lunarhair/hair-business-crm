# Hair Business CRM v5

У ZIP є дві версії:

1. `desktop.html` — версія для MacBook / ПК.
2. `mobile.html` — компактна версія для iPhone.

Що додано:
- Dashboard з ключовими KPI.
- Графіки: оборот, прибуток, витрати, кількість записів.
- База асортименту: товари / послуги / матеріали.
- Підтягування позицій з бази до клієнта.
- Матеріали по процедурі.
- Витрати по запису.
- Витрати бізнесу.
- Клієнтська база.
- Retention: кого запросити повторно.
- Топ процедур.
- Топ товарів.
- Monthly analytics.
- Delete / Undo.
- Export JSON.

Важливо для iPhone:
Локальні HTML-файли іноді погано відкриваються напряму в iOS.
Найкраще рішення: завантажити ці файли на GitHub Pages / Netlify / Vercel і відкрити `mobile.html` як сайт. Потім Safari → Share → Add to Home Screen.
